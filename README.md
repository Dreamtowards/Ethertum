## Getting Started

1\. [Install Rust](https://www.rust-lang.org/tools/install)

2\. Build & Run
```shell
git clone https://github.com/Dreamtowards/Ethertum.git
cd Ethertum

cargo run --release
# or
source build/client_debug.sh
```
